# Economic Model with LLM-president

## Идея проекта
Многоуровневая экономическая симуляция, где **домохозяйства и президент** принимают решения при помощи больших языковых моделей (LLM).
Остальные агенты действуют по классическим экономическим правилам.
Модель исследует, как сложность агентов и политические решения президента влияют на экономические показатели.
Для валидации полученной модели, будет проведено сравнение с классическими экономическими моделями.

- Агенты-домохозяйства и Президент могут работать в двух режимах: `rule-based` и `llm-based`.
